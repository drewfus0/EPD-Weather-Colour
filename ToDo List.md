### Current Displayed Information

**Header:**
- Date and Time (Day, Date, Month, Time)

**Current Weather (Top Section):**
- **Condition:** Icon + Text
- **Temperature:** Current + Feels Like
- **Wind:** Speed + Gust + Direction (Visual Arrow)
- **Atmosphere:** Humidity + Rain Probability
- **Other:** UV Index + Pressure

**Daily Forecast (Bottom Left - 3 Days):**
- Day Name
- Weather Icon
- High / Low Temperature
- Condition Text

**Hourly Graph (Bottom Right - 24 Hours):**
- **Axes:** Time (X), Temperature (Left Y), Rain % (Right Y)
- **Temperature Lines:** Forecast (Red) + History (Green)
- **Rain Bars:** Probability (Blue) + Actual (Orange)
- **Highlights:** Min/Max Dots & Labels for both Forecast and History
